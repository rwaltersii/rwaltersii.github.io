Projects

The projects posted here were assignments given to the class by the staff of MIT. This course work is in part, the work of the faculty and myself. The assignments were given with varying degress of 'starter code'. Addtional code was then added to complete the given exercise as instructed. Click on the "Launch App" to see the application in action. Clicking on the "Go to Repo" will open the GitHub repository where a copy of the code is stored.
